# MHD Custom

Marketing site and platform for MHD Custom — boutique custom cabinetry and fine
woodwork on Boston's North Shore. Designed, built, and maintained by
[Sqysh Studio](https://sqysh.io).

> No external contributions, forks, or pull requests are accepted.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS v4
- Zustand · Motion · Lucide
- Bebas Neue + Roboto via `next/font`
- Neon (Postgres) · Google OAuth
- Hosted on Vercel
